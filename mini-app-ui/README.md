# Language Learning Mini App

这是 `language-learning` 项目保留的小程序壳子工程，当前聚焦于登录、用户资料、请求封装、页面骨架和通用构建链路，不再包含历史打卡业务页面。

## 当前定位

- 保留 UniApp 基础工程结构
- 保留登录与用户信息获取链路
- 保留首页、我的页、设置页等基础入口
- 作为后续“语言学习”业务的小程序承载层

## 常用命令

```bash
pnpm install
pnpm type-check
pnpm dev:mp
pnpm build:h5
```

## 开发说明

- 首页和“我的”页已改为基础壳子页，可直接替换为新的学习业务模块。
- 请求层与用户状态管理仍保留，适合继续扩展课程、单词、学习计划等场景。
