# 项目概览

## 项目定位

`language-learning` 当前不是完整业务成品，而是面向后续“语言学习”项目的基础工程。目标是保留可复用的系统能力，移除历史打卡业务负担，便于新业务在干净基线上继续开发。

## 当前保留能力

- Web 端认证与会话管理
- 用户中心与个人资料维护
- 后台用户、角色、权限、菜单、配置、文件管理
- Prisma + PostgreSQL 数据层
- 阿里云 OSS、邮件、短信能力
- 微信小程序登录入口与小程序基础壳子
- 用户积分字段、VIP 字段、微信支付底层入口

## 已移除范围

- 全部打卡业务
- VIP 套餐、订单、奖金、题库、排行、优惠券等旧业务实现
- 与上述业务直接绑定的管理后台、API、小程序页面与初始化脚本

## 当前目录建议

- `app/`：Web 页面与 API
- `components/`：共享 React 组件
- `lib/`：认证、存储、配置、工具函数
- `prisma/`：数据库 schema、migration、seed
- `scripts/`：初始化与维护脚本
- `mini-app-ui/`：UniApp 小程序壳子
- `docs/`：当前文档；历史资料在 `docs/archive/`
