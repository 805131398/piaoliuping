# Markdown 编辑器使用说明

## 功能概述

这是一个基于网页的远程 Git 仓库 Markdown 文件编辑器，支持：

- 🔗 连接远程 Git 仓库
- 📝 在线编辑 Markdown 文件
- 👀 实时预览 Markdown 内容
- 💾 保存文件到本地仓库
- 🔄 Git 操作（提交、推送、拉取）
- 📁 文件浏览器和搜索

## 使用方法

### 1. 连接仓库

1. 访问 `/md-editor` 页面
2. 在配置面板中输入：
   - **仓库地址**：Git 仓库的 HTTPS 或 SSH 地址
   - **分支**：要编辑的分支（默认为 main）
   - **用户名**：GitHub 用户名（可选）
   - **访问令牌**：GitHub Personal Access Token（可选，用于私有仓库）

3. 点击"连接到仓库"按钮

### 2. 编辑文件

1. 在左侧文件浏览器中选择要编辑的 Markdown 文件
2. 在右侧编辑器中修改内容
3. 编辑器支持实时预览，可以同时看到编辑区和预览区
4. 修改后点击"保存"按钮保存到本地仓库

### 3. Git 操作

#### 提交更改
1. 在 Git 操作面板中输入提交信息
2. 点击"提交"按钮

#### 推送更改
1. 点击"推送更改"按钮将本地提交推送到远程仓库

#### 拉取更新
1. 点击"拉取更新"按钮从远程仓库获取最新更改

#### 检查状态
1. 点击"检查状态"按钮查看当前 Git 状态

## 支持的功能

### Markdown 编辑器
- 语法高亮
- 实时预览
- 工具栏支持常用 Markdown 语法
- 支持表格、代码块、链接等

### Git 操作
- 克隆仓库
- 切换分支
- 提交更改
- 推送/拉取
- 状态检查

### 文件管理
- 文件浏览器
- 文件搜索
- 文件夹展开/折叠
- 文件修改状态显示

## 技术栈

- **前端**：Next.js 15, React 19, TypeScript
- **UI 组件**：Shadcn UI, Radix UI
- **Markdown 编辑器**：@uiw/react-md-editor
- **Git 操作**：simple-git
- **样式**：Tailwind CSS

## 注意事项

1. **权限要求**：确保有仓库的读取和写入权限
2. **私有仓库**：需要提供 GitHub Personal Access Token
3. **大文件**：建议不要编辑过大的文件
4. **并发编辑**：避免多人同时编辑同一文件
5. **网络连接**：需要稳定的网络连接进行 Git 操作

## 故障排除

### 连接失败
- 检查仓库地址是否正确
- 确认网络连接正常
- 验证访问权限

### 保存失败
- 检查文件权限
- 确认磁盘空间充足
- 验证文件路径有效

### Git 操作失败
- 检查 Git 配置
- 确认远程仓库可访问
- 验证提交信息格式

## 开发说明

### 项目结构
```
app/md-editor/
├── page.tsx              # 主页面
├── components/
│   ├── FileExplorer.tsx  # 文件浏览器
│   └── GitManager.tsx    # Git 操作管理器
└── api/
    ├── connect/          # 连接仓库
    ├── file/             # 文件操作
    └── git/              # Git 操作
```

### 本地开发
1. 安装依赖：`npm install`
2. 启动开发服务器：`npm run dev`
3. 访问：`http://localhost:3000/md-editor`

### 部署注意事项
- 确保服务器有足够的磁盘空间存储仓库
- 配置适当的文件权限
- 考虑使用环境变量管理敏感信息
