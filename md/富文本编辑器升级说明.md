# 富文本编辑器升级说明

## 概述

本次升级将 MD Editor 从基础的 `@uiw/react-md-editor` 升级为功能强大的 `reactjs-tiptap-editor`，提供了更丰富的编辑体验和功能。

## 主要改进

### 1. 新增组件
- **RichMarkdownEditor.tsx** - 基于 demo 目录实现的新富文本编辑器组件
- 保持与原有 MD Editor 架构的兼容性
- 支持与 Git 管理、文件浏览器的无缝集成

### 2. 功能增强

#### 编辑功能
- ✅ **所见即所得编辑** - 实时预览，无需切换视图
- ✅ **丰富的格式支持** - 标题、粗体、斜体、代码块、表格等
- ✅ **扩展功能** - 表情符号、数学公式(Katex)、Mermaid图表
- ✅ **文件上传** - 支持图片、视频、附件上传
- ✅ **绘图工具** - 内置 Excalidraw 绘图和简单绘图工具
- ✅ **多列布局** - 支持复杂的文档布局
- ✅ **任务列表** - 支持嵌套的待办事项

#### 用户体验
- ✅ **主题切换** - 支持亮色/暗色主题切换
- ✅ **全屏模式** - F11 快捷键支持
- ✅ **快捷键支持** - Ctrl+S 保存等常用快捷键
- ✅ **气泡菜单** - 选中文本时的快捷工具栏
- ✅ **斜杠命令** - 快速插入各种元素
- ✅ **搜索替换** - 内置搜索和替换功能

#### 导入导出
- ✅ **Word 导入/导出** - 支持 docx 格式
- ✅ **PDF 导出** - 直接导出为 PDF
- ✅ **Markdown 兼容** - 完全兼容 Markdown 语法

## 使用指南

### 基本操作
1. **选择文件** - 从左侧文件浏览器选择 Markdown 文件
2. **开始编辑** - 在富文本编辑器中直接编辑
3. **保存文件** - 使用 Ctrl+S 或点击保存按钮
4. **切换主题** - 点击工具栏的主题切换按钮

### 高级功能
1. **插入表格** - 使用工具栏或斜杠命令 `/table`
2. **添加公式** - 斜杠命令 `/katex` 插入数学公式
3. **绘制图表** - 斜杠命令 `/mermaid` 创建流程图
4. **上传图片** - 直接拖拽或使用图片按钮

### 快捷键
- `Ctrl+S` - 保存文件
- `F11` - 全屏切换
- `Ctrl+Z` - 撤销
- `Ctrl+Y` - 重做
- `/` - 打开斜杠命令菜单
